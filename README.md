
<h1 align="center">HoloSolv-AI</h1>

> ✍️ Draw in the Air. Let AI Solve It.

HoloSolv-AI is an AI-powered Computer Vision application that allows users to write mathematical expressions and draw geometric figures in the air using hand gestures. The system captures the drawing through a webcam, processes it using Computer Vision, and leverages Google's Gemini AI to generate solutions with detailed explanations.

---

## ⚡ Features

- 🖐️ Real-time hand tracking
- ✍️ Air-writing interface
- 🧮 Mathematical expression solving
- 🤖 AI-powered reasoning
- 📐 Geometric shape recognition
- 📊 Area, perimeter, and angle calculations
- ⚡ Instant result generation
- 🎓 Interactive learning experience

---

## 🛠️ Tech Stack

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/HoloSolv-AI.git
cd HoloSolv-AI
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure Environment Variables

Create a `.env` file:

```env
GOOGLE_API_KEY=YOUR_ACTUAL_API_KEY
```

### Run the Application

```bash
streamlit run app.py
```
